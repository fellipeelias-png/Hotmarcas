# HotMarcas — Notas do projeto

## Preferências do usuário

- **Deploy automático ao terminar:** sempre commitar e fazer push para o branch de trabalho ao final de cada tarefa, sem perguntar. O push para o remoto dispara o deploy na Vercel (configurado em `vercel.json`).
- **Linguagem direta, sem juridiquês:** evitar termos como "classes NCL" ou citar números de classe da Classificação de Nice no conteúdo voltado ao cliente final. Preferir frases diretas, em português comum.
